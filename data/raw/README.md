## Raw Data Description

This folder contains the raw data inputs used in the bibliometric analysis.

# Files
- `rayyan.csv`  
  Rayyan screening output containing records included after title and abstract
  screening.

- `WOS_2019_2020.bib`  
- `WOS_2021_2022.bib`  
- `WOS_2023_2024.bib`  
- `WOS_2025.bib`  

These files are bibliographic records exported from the Web of Science Core
Collection in BibTeX format.

# Data Access Note
Web of Science data are subject to licensing restrictions and cannot be publicly
redistributed outside academic use. Users wishing to reproduce the analysis
should retrieve the data directly from the Web of Science database following
the search strategy described in the dissertation.
